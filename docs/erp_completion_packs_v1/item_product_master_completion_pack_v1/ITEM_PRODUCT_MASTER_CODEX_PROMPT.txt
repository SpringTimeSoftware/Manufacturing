MISSION:
Make Item/Product Master commercially credible and testable.

NON-NEGOTIABLE GATES:
1. Category, Subcategory, Product Family, Business Segment, Reporting Bucket, UOM and all master-linked item fields must be lookup/select/disabled governed selectors, never unrestricted text.
2. Net Weight, Gross Weight, Dimensions, Conversion Factor, MOQ, Lead Time, Prices, Discounts, Tax, Capacity and all measurable item values must use numeric/decimal/money controls, never unrestricted text.
3. Upload image/media must be implemented as item-specific upload/list/set-primary/retire with authorization, or hidden/disabled with reason. Do not show fake upload.
4. Save Draft / Save & Continue / Edit Existing / Activate/Inactivate / View Audit / Related links must be working, disabled with reason, or hidden.
5. Save/reopen must preserve values from all item sections, not only Core Info.
6. No right-drawer deep editor.
7. No internal/scaffold wording.
