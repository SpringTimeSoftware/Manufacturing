GLOBAL HARD RULES

1. No visible touched action may remain dead.
2. Every visible touched action must end as:
   - WORKING
   - DISABLED WITH REASON
   - HIDDEN
3. Desktop BOM/routing/ECO line entry must use compact editable grids/tables.
4. Card-per-line desktop line entry is a failure condition.
5. No line editor may save only components[0], operations[0], or first line.
6. No governed field may remain unrestricted free text where source exists.
7. No numeric/decimal/time/quantity field may remain unrestricted text.
8. Released BOM/routing/document revisions cannot be directly editable.
9. Engineering changes to released objects must go through clone/revision/ECO or be disabled with reason.
10. Upload/document actions must work or be disabled with reason.
11. Linked-record actions must open exact records or be disabled with reason.
12. No internal/scaffold wording in production-facing UI.
