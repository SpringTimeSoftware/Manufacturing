# Procure-to-Pay Completion Pack V1

This pack is the detailed completion contract for the Manufacturing ERP procure-to-pay domain.

Files:
- `PROCURE_TO_PAY_BENCHMARK_V1.xlsx`
- `PROCURE_TO_PAY_COMPLETION_SPEC.md`
- `PROCURE_TO_PAY_CODEX_PROMPT.txt`

Run this pack after:
1. Item/Product Master
2. Customer/Dealer/Distributor Master
3. Supplier/Vendor Master
4. Quote/Sales Order/Forecast/ATP

This pack covers:
- Purchase Requisition
- RFQ
- Supplier Quotation
- Quote Comparison
- Purchase Order
- Subcontract Purchase Order / Outside Processing
- Goods Receipt Note / GRN
- Purchase Invoice / 2-way and 3-way match
- Vendor Return
- Landed Cost
- Payment Schedule / AP handoff
- Procurement dashboards and buyer queues

Strict rule:
No P2P transaction screen is complete unless it supports multiline where applicable, add/remove/edit all lines, save/reopen, correct lookup/numeric controls, action truth, screenshots, and validation.
