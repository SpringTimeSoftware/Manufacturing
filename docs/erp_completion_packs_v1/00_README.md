# ERP Completion Packs V1

This pack contains three detailed completion packs created in the same style as the accepted Item/Product Master pack:

1. Customer / Dealer / Distributor Master
2. Supplier / Vendor Master
3. Quote / Sales Order / Forecast / ATP

Each folder contains:
- BENCHMARK_V1.xlsx: source-backed benchmark, target fields, action contract, workflow contract, tests, anti-patterns, mapping/gap templates
- COMPLETION_SPEC.md: module-specific completion definition
- CODEX_PROMPT.txt: strict Codex execution prompt

Use these sequentially:
1. Customer / Dealer / Distributor Master
2. Supplier / Vendor Master
3. Quote / Sales Order / Forecast / ATP

Do not accept a module as complete unless its P0/P1 tests pass, screens show correct controls/actions, and review screenshots are produced.
