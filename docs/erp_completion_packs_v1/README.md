# ERP Remaining 8 Completion Packs v1

This folder contains the remaining 8 detailed completion pack folders, grouped into two 4-pack waves but designed to be executed one pack at a time.

## Start Here

1. Read `00_EXECUTION_STRATEGY_README.md`.
2. Run `02_CROSS_PACK_RESIDUAL_GAP_CLOSURE_AUDIT.md` as an emergency check if quote/SO salesperson, remarks, bin selection, revisions, price/discount, or tax remain doubtful.
3. Use `03_WAVE_1_PROMPT_PACKS_04_TO_07.md` and `04_WAVE_2_PROMPT_PACKS_08_TO_11.md` for wave-level orientation only.
4. Use each pack's `codex_prompt.md` for real implementation.
5. Require Codex to fill `codex_output_report_template.md` after every pack.

## Recommended Implementation Style

Run one pack at a time. Broad multi-pack implementation prompts are useful for planning but risky for deep coding because they hide gaps.
